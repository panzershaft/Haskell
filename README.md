# Haskell

Will be using http://learnyouahaskell.com/ as reference for learning Haskell.  
Various problems will also be added with solutions in Haskell. 

To execute a functitons:  
* Make a fucntion ex demo.hs
* In the shell with ghci compiler activated run
  ```
  :l demo
  ```
* Call the function (addMe is a function inside demo.hs)
  ```
  addMe 2 5
  7
  ```
